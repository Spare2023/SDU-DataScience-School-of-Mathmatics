# 实验1：MATLAB 基础操作

> 📌 本文件夹仅含源码 `.m`。运行所需输入图片与第三方工具箱不在此目录，需从原项目根目录获取。

## 文件清单

### `experiment1.m`
- **作用**：完成 MATLAB 基础操作练习，包括：① 设置搜索路径（`addpath`/`genpath`）；② 矩阵运算（生成 5 阶魔方矩阵、子矩阵索引、矩阵乘法 `b*c` 与逐元素乘法 `b.*c`）；③ 用 `meshgrid` 生成 101×161 的圆形矩阵 A（圆上=0.2，其余=0.8）并可视化；④ 用 `save`/`load` 保存与重新载入工作空间变量。
- **使用方法**：在 MATLAB 中直接运行 `experiment1.m` 即可。脚本会自动在自身所在目录创建 `figs/` 子目录并保存圆形矩阵可视化图。注意：任务1 中写死了路径 `D:\MATLAB\vision\myDemo`，如需实际保存可改该路径或忽略（仅影响保存位置）。
- **依赖**：仅基础 MATLAB，无需额外工具箱。

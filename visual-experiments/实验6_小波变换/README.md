# 实验6：多尺度分析与小波变换

> 📌 本文件夹仅含源码 `.m`。运行所需输入图片不在此目录，需从原项目根目录获取。

## 文件清单

### `experiment6.m`
- **作用**：基于小波变换的图像处理。① Symlets 三尺度分解/重构做多方向边缘检测；② 加噪 lena 小波去噪（全局/分层/贝叶斯阈值），输出 PSNR/SSIM；③ Biorthogonal 三尺度渐进重构；④ 构造四级小波金字塔。
- **使用方法**：直接运行。需同目录提供 `house.png`、`lena.png`。
- **依赖**：MATLAB Wavelet Toolbox。
